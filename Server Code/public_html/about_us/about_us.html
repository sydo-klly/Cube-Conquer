CubeConquer

How to play:

1: Someone must download the executable and create a gameCode

2: Players should join the game through here (we recommend using a smartphone)

3: When all players have joined, press Start Game

4: Each player will start with 10 units in one of the bases in the map

5: Every 5 seconds you get +2 units in your bases and you can send troops to other ones

6: If you are sending less than 100 units your troops will travel in a HUMVEE or Tank which can only move to adjacent bases
but if sending more your troops will fly in a transport helicopter and can travell to the other side of the map in just seconds
